How LyricalWindows Works

  1. 1. Type the artist and song title.
  2. 2. Paste a YouTube or Spotify track link — it loads as a real, playable background track (no downloads, no server needed). Spotify tends to actually play more reliably, since YouTube blocks embedding on a lot of videos.
  3. 3. Auto-fetch the lyrics (or type them in), then tag each line with a timestamp and a mood (love, sad, angry, quiet, beatdrop, happy).
  4. 4. Hit Play — retro OS-style windows pop up in sync, each themed and animated to match the mood of that line.
