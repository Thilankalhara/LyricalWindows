# 🪟 LyricalWindows

A retro OS-inspired synced lyrics player that displays animated
Windows-style popups in perfect sync with your music.

Paste a **YouTube** or **Spotify** link, fetch synced lyrics, and enjoy
a nostalgic desktop experience with Windows 95, Windows XP, macOS Glass,
and Windows 10 themes.

## ✨ Features

-   🎵 Real background music from YouTube or Spotify
-   📝 Auto-fetch synced lyrics from LRCLIB
-   💻 Windows 95 theme
-   💙 Windows XP theme
-   🍎 macOS Glass theme
-   🪟 Windows 10/11 theme
-   ❤️ Mood-based popup animations
-   ⚡ Beat drop effects
-   🎉 Happy & emotional lyric animations
-   🌐 100% browser-based
-   🚀 Deployable on GitHub Pages

## 🛠️ How It Works

1.  Enter the artist name.
2.  Enter the song title.
3.  Paste a YouTube or Spotify URL.
4.  Fetch synced lyrics.
5.  Click **Start**.
6.  Enjoy synchronized retro popup windows.
